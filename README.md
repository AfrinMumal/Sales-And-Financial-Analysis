# Sales & Financial Analysis Dashboard (Excel)

## Project Overview
This project presents a Sales and Financial Performance Analysis Dashboard built using Microsoft Excel. The dashboard analyzes the sales performance of AtliQ Hardware across customers, markets, products, and divisions.

The goal of this project is to transform raw business data into meaningful insights that support data-driven decision making.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization
- Conditional Formatting
- Business Performance Analysis

## Key Analysis Performed

### 1. Profit & Loss Analysis
- Net Sales
- Cost of Goods Sold (COGS)
- Gross Margin
- Gross Margin %

Quarterly performance from 2019 to 2021 is analyzed to understand financial trends.

### 2. Customer Performance Analysis
Evaluation of sales performance by different customers such as:
- Amazon
- Flipkart
- Reliance Digital
- Croma

The dashboard also compares growth between 2021 vs 2020.

### 3. Market Performance vs Target
This section compares actual sales performance against targets across different countries to identify underperforming markets.

### 4. Division Performance
Performance comparison of divisions such as:
- N & S
- P & A
- PC

Growth percentage from 2020 to 2021 is analyzed.

### 5. Product Performance
The dashboard highlights:
- Top 5 performing products
- Bottom 5 performing products
- Newly launched products in 2021

### 6. Country-wise Sales Analysis
Top performing countries based on sales revenue are identified.

## Insights Generated
- Identification of high revenue customers
- Market performance gaps vs targets
- Product level performance analysis
- Division growth comparison
- Country-wise revenue contribution
